Sistema para control y gestion de componentes
Cas-Cam/LV
Desarrollado para Compañia Minera Doña Ines de Collahuasi SCM



Este sistema esta diseñado para llevar el control de los componentes Cas-Cam
para Vehiculos livianos que han sido, o se encuentran instalados, en la flota
de vehiculos menores que se encuentra en circulacion por circuitos interior mina
y que cuentan con este estandar en su totalidad.

Este sistema se encuentra en etapa de desarrollo, por lo tanto,
cualquier critica o aporte para el sistema, sera muy bien recivida, con la finalidad de poder
hacer un sistema mas amigable y potente, con el objetivo de obtener una version estable y final, favor
enviar sus criticas u aportes via correo a pffiguer@collahuasi.cl
